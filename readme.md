Optimized for `prettier`.

`react`, `flowtype`, `prettier`.

## Install

```
$ yarn add --dev eslint eslint-config-precure
```

```
$ npm install --save-dev eslint eslint-config-precure
```

## Usage

.eslintrc

```
{
  "extends": "precure"
}
```

### OSS:

`flow` && `jest`

```
{
  "extends": ["precure/oss"]
}
```
